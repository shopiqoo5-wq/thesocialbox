# The Social Box

The Social Box is a modern agency website built with React, Vite, and Tailwind CSS.

## Features

- Fully responsive design
- Modern UI/UX with polished animations
- Multiple pages: Home, About, Services, Impact, and Case Studies

## Tech Stack

- **Frontend**: React.js
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Routing**: React Router
- **Build Tool**: Vite

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/project-x-sumit/thesocialbox.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
